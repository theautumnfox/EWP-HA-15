Um ein git-Repository zu klonen, verwendet man den Befehl git clone. Mit git add wird die Zwischenspeicherung vorbereitet und mit git commit durchgeführt. Mit dem Befehl git push werden die Änderungen hochgeladen. Um das eigene Arbeitsverzeichnis mit dem Repositoy zu synchonisieren verwendet man den Befehl git pull
